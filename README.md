### Environment
- Ubuntu LTS

### Technologies
`Express.js` | `Node.js` | `mongodb` | `nodemailer`

### Set up environment
1. Clone the following project repo
- `git clone repository-link`
2. Navigate to the project folder
- `cd nanny-hub-api`
3. Install libraries
- `npm install`
4. Start server
- `npm run dev`

*Server should run on `http://localhost:8000`*
